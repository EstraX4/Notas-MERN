
# Comparación con el Operador de Desigualdad

El operador de desigualdad ( `!=`) es lo opuesto al operador de igualdad. Significa no igual y vuelve `false`donde volvería la igualdad `true`y _viceversa_ . Al igual que el operador de igualdad, el operador de desigualdad convertirá tipos de datos de valores al comparar.

**Ejemplos**

```js
1 !=  2    // true
1 != "1"   // false
1 != '1'   // false
1 != true  // false
0 != false // false
```
